Storage configuration simplified.

Blivet is a free & open source python module for configuring system storage on Linux. It provides powerful modeling capabilities along with a simple, seamless interface for managing all of your favorite storage technologies. Learn more.

Features

  • Modeling of changes before committing to disk
  • Easy configuration of complex device stacks using a top-down specification
  • Powerful disk partitioning functionality

Supported Storage Technologies

  • LVM
  • MD
  • LUKS
  • XFS
  • EXT4
  • BTRFS
  • Firmware RAID
  • SAN
  • Mainframe
